# Настройка Web HTTP

Легковесный интерфейс `http` предоставляет агенту навыки для отправки сырых HTTP-запросов (`GET`, `POST` и др.) и прямого скачивания файлов в папку `sandbox/_system/download/`. Отлично подходит для работы с кастомными API и REST-сервисами, которые не требуют поднятия тяжелого браузера.

## Параметры (`web.http`)

* **`enabled`**: `true` / `false`.
* **`request_timeout_sec`**: Таймаут (в секундах) на ожидание ответа от сервера.
* **`max_response_chars`**: Жесткий лимит на количество символов в теле ответа. Если сервер вернет гигантский JSON на 10 МБ, система автоматически обрежет его, чтобы не вызвать ошибку `Context Window Exceeded` у языковой модели.
