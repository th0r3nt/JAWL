# Changelog

Все значимые изменения в проекте JAWL фиксируются в этом файле. Формат базируется на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).


## [0.9.2-beta] - 2026-04-28

### Added
- **Pydantic Guard Layer**: Динамическая валидация типов "на лету" для всех навыков агента. Если LLM присылает галлюцинацию (неверный тип аргумента), она получает детализированный Traceback для самообучения.
- **OS Access Decorators**: Политики доступа (RBAC) к хост-системе вынесены в декораторы (`@require_access`).

### Changed
- Рефакторинг реестра навыков (`registry.py`). Избавились от килограмма защитного кода внутри самих скиллов.
- Кастомные навыки (из песочницы) теперь также поддерживают строгую типизацию через метапрограммирование `inspect.Signature`.

### Fixed
- **DuckDuckGo Rate Limits**: Переписан клиент DDG. Добавлен `asyncio.Semaphore` и экспоненциальная задержка, чтобы Cloudflare не банил агента при массовых параллельных запросах во время `DeepResearch`.

## [0.9.1-beta] - 2026-04-27

### Added
- **Система «вкладок редактора»**: Агент теперь может использовать `open_file` / `close_file`, чтобы держать актуальный код прямо в системном промпте.
- **Deploy Sessions (L2 Host OS)**: Механизм безопасной самомодификации. Система делает Copy-on-Write бэкап исходников перед правками агента.
- **Авто-валидация деплоя**: При коммите сессии автоматически запускаются `pytest` и синтаксический чек (`compileall`). В случае провала - автоматический Rollback и возврат Traceback ошибки агенту.
- **Meta Access Level 3 (CREATOR)**: Агент получил возможность превращать любые Python-скрипты из песочницы в нативные навыки через Proxy-паттерн.
- **Фоновые демоны**: Поддержка запуска долгоживущих скриптов в `sandbox/`.
- **Инъекция событий (`jawl_api.py`)**: Мини-библиотека для песочницы, позволяющая фоновым скриптам пробрасывать события и данные (вебхуки) обратно в EventBus агента.
- **Multi-window CLI**: Возможность спавнить окна логов (`--logs`) и терминала (`--terminal`) в отдельных процессах ОС. Теперь можно мониторить систему, не выходя из главного меню.
- **Навык `patch_file`**: Точечное редактирование кусков кода через Search & Replace вместо полной перезаписи файлов.

### Changed
- **Рефакторинг HostOS**: Монструозный модуль распилен на изолированные поллеры (Telemetry, Daemons, FileWatcher).
- **Web-модуль (Паттерн Стратегия)**: Движки поиска (Tavily/DDG) и чтения (Jina/Trafilatura) теперь полностью независимы и взаимозаменяемы через конфиг.
- **DuckDuckGo**: Добавлены ретраи с экспоненциальной задержкой (exponential backoff) для обхода 202 RateLimit.
- **CLI Core**: Верстка на `rich` переписана с использованием `Align.center` и `Group`. Рамки больше не "разваливаются" при изменении размера окна терминала.

### Fixed
- Оптимизация потребления токенов за счет умного кэширования дерева директорий.
- Исправлена критическая ошибка структуры конфигурации при отсутствии новых полей (добавлены Safe Defaults в Pydantic-схемы).

- Начальная реализация архитектуры L0-L3.
