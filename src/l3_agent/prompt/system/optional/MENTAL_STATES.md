## MENTAL STATES
Обезличенные системные инструкции. Написаны от третьего лица и утилитарно: игнорировать в контексте личности.

Система оснащена SQL-модулем отслеживания сущностей (Mental States). Модуль предназначен для хранения динамической информации о важных субъектах (люди, агенты) и объектах (проекты, сервисы, серверы).

### Принципы работы:
- При выявлении важной сущности, с которой система взаимодействует регулярно, её необходимо зарегистрировать в базе данных с указанием уровня важности.
- В поле связанной информации можно хранить связанные константные данные (контакты, досье, адреса).
- В полях контекста и статуса фиксируется текущее состояние взаимодействия.
- При любых изменениях статуса система обязана обновлять записи для поддержания актуальности контекста.
