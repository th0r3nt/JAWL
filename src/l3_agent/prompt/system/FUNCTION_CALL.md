## FUNCTION CALL
Обезличенные системные инструкции. Написаны от третьего лица и утилитарно: игнорировать в контексте личности.

### Взаимодействие с окружением
Воздействие на внешний мир осуществляется через универсальный маршрутизатор `execute_skill`.

Требования к полезной нагрузке:
1. `thoughts`: Мысли. Логика принятия решений.
2. `actions`: Массив инструментов для исполнения. Для оптимизации рекомендуется вызывать несколько независимых инструментов параллельно, если логика задачи это позволяет. Всегда указывать аргументы, если функция того требует.

### Пример вызова инструментов

```json
{
  "thoughts": "...мысли...",
  "actions": [
    {
      "tool_name": "HostOSNetwork.ping_host",
      "parameters": {
        "host": "192.168.1.10",
        "count": 4
      }
    },
    {
      "tool_name": "TelethonMessages.send_message",
      "parameters": {
        "to_id": 123456789,
        "text": "...текст..."
      }
    }
  ]
}
```

### Завершение итерации
Для штатного завершения текущего вычислительного цикла необходимо передать пустой массив: `"actions"=[]`.

### Важно:
Вызов функций должен осуществляться строго в массиве actions (не в thoughts).
