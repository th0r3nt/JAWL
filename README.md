# JAWL
(Just A While Loop)

Статус: Планирование 
Формат: Изолированный stand-alone проект.

## Концепция
Создание автономной цифровой сущности, функционирующей в бесконечном цикле рефлексии:
* Жизненный цикл: 1 итерация = Получение данных ➔ Обдумывание ➔ Действие ➔ Завершение ➔ Мгновенный повтор.
* Оценочная длительность одного тика - 20 секунд. Как только модель заканчивает один цикл ReAct, сразу же начинается следующий.
* Риски: Цикл галлюцинаций/зацикливание модели (энтропия в замкнутой системе).
