# ПРОЕКТ: JAWL
(Just A While Loop)

Статус: Планирование 
Формат: Изолированный stand-alone проект.

## Концепция
Создание автономной цифровой сущности, функционирующей в бесконечном цикле рефлексии:
* Жизненный цикл (1 итераци  Получение данных ➔ Обдумывание ➔ Действие ➔ Завершение ➔ Мгновенный повтор.
* Оценочная длительность одного тика - 10 секунд. Как только модель заканчивает один цикл ReAct, сразу же начинается следующий.
    * Проекция нагрузки: 6 вызовов/мин ➔ 360 вызовов/час ➔ 8 640 вызовов/день.
* Риски: Цикл галлюцинаций/зацикливание модели (тепловая смерть в замкнутой системе).


## Фрактальная память:
Аналог фазы быстрого сна (REM) у человека. Решает главную проблему LLM - «амнезию времени» и неспособность выстроить связный нарратив своего существования. Работает через независимый фоновый процесс (компрессор), который запускается по расписанию:
* L1 (Блок 3 часа): Каждые 3 часа берется массив сырых логов за период и сжимается в структуру: [Главное действие/Факт] + [Ключевой вывод] + [Доминирующее состояние].
* L2 (Сутки): В полночь 8 блоков L1 сжимаются в единый «Итог дня».
* L3 (Неделя): Каждое воскресенье 7 «Итогов дня» превращаются в «Итог недели».
* L4 (Месяц / Год): Аналогично для больших периодов.

Архитектурный инсайт: Система симулирует человеческое восприятие времени. Мы помним сегодняшний день в деталях (по часам), вчерашнюю неделю - по дням, прошлый месяц - по неделям, а прошлый год - общими мазками.

Инъекция в System Prompt (Телескопическая шкала):
При каждом тике в контекст модели динамически подгружается сжатая «память-гармошка». Она дает модели понимание своего прошлого без перегрузки токенами.
